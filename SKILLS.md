기타
- 외부 라이브리는 항상 오픈소스를 우선적으로 선택한다.
- 외부 라이브러리는 항상 최신 버전을 사용한다.
- 외부 라이브러리는 항상 문서화가 잘 되어 있는 것을 선택한다.

- 외부 라이브러리는 항상 라이센스가 자유롭게 사용 가능한 것을 선택한다.
- 외부 라이브러리는 CDN을 사용하지 않고 내부에 다운로드해서 사용한다.
